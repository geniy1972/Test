Компания:

Клиент представляет собой онлайн видео рекламную платформу, которая позволяет рекламодателям повышать свою производительность за счет предоставления рекламы для конкретной аудитории, акцентируясь на целях рекламадателя.

PHP

Проект:

Система изначально была создана на PHP/JavaScript/jQuery. Сейчас команда выполняет рефакторинг из ключевых компонентов системы, а также занимается развитием новых модулей, использующих Java / JavaScript / Angular JS. Мы хотим создать приложение, с компонентами не сильно связанными между собой иMVC patterns c хорошей структурой кода, с UI и  разделенной бизнес-логикой.
